## Compiling and running
Run `make`. An executable will compile. Runs on OS X.

Run it with choose one . `./fractol [mandelbrot / julia / duobrot / triobrot / ship]`. <br>
Example . `./fractol mandelbrot`. <br>
Or . `./fractol julia`. <br>
