## Compiling and running
Run `make`. An executable will compile. Runs on OS X.

Run it with `./fdf test/map.fdf`.
`scroll on mouse` -- Zoom in or out.
`<` `>` -- Change altitude.
`NUM0-NUM9` -- Change colors theme.
